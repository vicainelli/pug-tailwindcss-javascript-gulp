### Install dependencies

`yarn`

### Run Build Task
`npm run production`

### Run Server Task (watch)
`gulp server`

### Run Individual Tasks

#### CSS (TailwindCSS)
`gulp css`

#### Pug
`gulp views`

#### Copy Images
Just to copy Images from `src` to `dist`
`gulp copy-images`

### Run Build Task
`npm run production`
