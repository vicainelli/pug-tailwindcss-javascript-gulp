# Pug + Sass + Javascript + Gulp

[![license](https://img.shields.io/github/license/vicainelli/pug-tailwindcss-javascript-gulp.svg)](LICENSE)

### Install dependencies

`yarn`

### Run Individual Tasks

#### CSS (TailwindCSS)
`gulp css`

#### Pug
`gulp views`


### Run Watch Task
`gulp`

### Run Build Task
`npm production`
